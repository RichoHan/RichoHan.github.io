MediaGoblin 其他相關指令

Date 六 30 11月 2013 By Richo Category Others Tags MediaGoblin

當你環境設定出現更改之後，可以跑一下指令來更新 

./bin/python setup.py develop --upgrade && ./bin/gmg dbupdate

如果MediaGoblin原始開發者有更動程式碼，用以下指令來更新程式碼 

git pull -u  
git submodule update

一次清除所有使用者資料 

rm -rf mediagoblin.db kombu.db celery.db user_dev; ./bin/gmg dbupdate
