MediaGoblin 佈署

Date Sat 30 November 2013 By Richo Category Others Tags MediaGoblin

前言

初始的佈署方式都被寫在mediagoblin.ini檔裏面，如果你要更動設定，建議先將mediagoblin.ini複製到mediagoblin_local.ini，再直接改寫mediagoblin_local.ini內的設定即可，預設就是如果有mediagoblin_local.ini就會優先去讀它。

1.複製mediagoblin.ini至mediagoblin_local.ini

cp mediagoblin.ini mediagoblin_local.ini

2.常見更動

一開始由於email系統其實並沒有開啟，方便開發者測試帳號註冊，所以當使用者註冊完帳號之後，需要去點terminal的連結來開啟帳號服務，當我們想要開啟對外服務之後可以做以下更改： 

email_debug_mode = false

然後針對系統自動發信地址做設定： 

email_sender_address = &#34foo@example.com&#34 #這部份看你的domain name是什麼

Reference:Configuration MediaGoblin

paste.ini的更動方式如同mediagoblin.ini

1.複製paste.ini至paste_local.ini

cp paste.ini paste_local.ini

2.如果想對外開啟服務，可能就不能只是在local端測試了，這時要該改ip和port設定

直接更改 

[server:main]  
use = egg:Paste#http  
host = site.domain || ip  
port = port number

存檔之後重新跑server就可以對外服務了
